Delta Memory Force V2.0

 

În cazul în care aveţi pachetul de instalare în formă arhivată, trebuie să rulaţi fişierul Setup.exe din interiorul arhivei.

Dacă nu aveţi programul WinZip,WinAce,WinRAR,  puteţi folosi DMF_KIT.exe. CD-ul conţine pachetul de instalare în mai multe variante de comprimare:

 

 

Ø       DMF_KIT.zip

Ø       DMF_KIT.rar

Ø       DMF_KIT.ace

Ø       DMF_KIT.exe

Ø       DMF_KIT.cab

Ø       DMF_KIT.tar

Ø       DMF_KIT.lzh

Ø       DMF_KIT.jar

 

 

CD-ul conţine pachetul de instalare şi sub forma unor fişiere comprimate de 1.44 Mb pentru “Floppy Disk”.Programul şi fişierele aferente din aceste arhive trebuiesc doar copiate, nefiind necesară instalarea programului.Descărcarea pachetelor de instalare de 1.44Mb se poate face mai jos:

 

 

Ø       1.44Mb.exe

Ø       1.44Mb.ace

Ø       1.44Mb.zip

Ø       1.44Mb.rar
